1. install uv in the system
2. go to folder where you want to create virtual env.
3. use the command
   ```
   uv init
   ```
4. now the uv env creation set-up will be created and default python version will be global python version
5. if you want to change the default python version, use the command
   ```
   uv python install <version>
   ```
   to remove the installed specfic version go to "users/username/.local/bin" and delete the python file

6. create the virtual env
   ```
   uv venv
   ```
   or
   ```
   uv venv â€“-python <version>
   ```
   or
   ```
   uv venv <name> --python <version>
   ```
7. activating environment with mac/linux
   ```
   source .venv/bin/activate
   ```
   with windows:
   ```
   .venv\Scripts\activate.bat
   ```

8. adding dependencies
   ```
   uv add <>
   ```
   with custom name: 
   ```
   uv --active add <>
   ```
9. running jupyter notebook
   ```
   uv run jupyter jupyter lab
   ```
   or
   ```
   uv run --active --with jupyter jupyter lab
   ```
